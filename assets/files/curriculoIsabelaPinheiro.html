DESENVOLVIMENTO DE UMA APLICAÇÃO MÓVEL EM UM AMBIENTE DE NEGÓCIO

  • Função no projeto: desenvolvedora;
  • Nome do projeto: Why Park;
  • Início: agosto/2022 | Término: dezembro/2022;
  • Descrição: desenvolvimento de uma aplicação móvel em um ambiente de negócio para a disciplina de “Projetos” do terceiro período do curso de Análise e Desenvolvimento da PUC-MG. Para o desenvolvimento foram utilizadas as seguintes tecnologias: React Native, JavaScript e Firebase. Por meio do aplicativo foi possível realizar cadastro, login, buscas, reservas de vagas e avaliações de acordo com a experiência do usuário em cada empresa de estacionamento.

DESENVOLVIMENTO DE UMA APLICAÇÃO WEB FRONT END

  • Função no projeto: desenvolvedora;
  • Nome do projeto: Web Info Legislativo;
  • Início: agosto/2021 | Término: dezembro/2021;
  • Descrição: criação de uma aplicação web front-end para a disciplina de “Projetos” do primeiro período do curso de Análise e Desenvolvimento de Sistemas da PUC-MG. Para o desenvolvimento foram utilizadas as seguintes tecnologias: HTML, CSS e JavaScript. O projeto Web Info Legislativo consistiu na criação de um site que informava sobre atualizações legislativas. Para o armazenamento de dados, foi criado um banco de dados fictício, utilizando o JavaScript. Por meio do site, era possível visualizar legislações na versão atualizada e na versão antiga. Além disso, havia possibilidade de busca por data, palavra-chave ou número de lei. Por fim, também era possível fazer o download das legislações em formato .pdf ou .docx.
